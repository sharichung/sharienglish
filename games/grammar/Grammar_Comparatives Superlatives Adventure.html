Comparatives & Superlatives Guide

Look for the word pattern first. Short words, y-ending words, and long words use different changes.

1-syllable: add -er / -est

Most short adjectives just add a new ending.

tall -> taller -> tallest fast -> faster -> fastest

Short vowel + final consonant: double it

Some short words need the last consonant doubled before the new ending.

big -> bi + ger -> bigger big -> bi + gest -> biggest hot -> hotter -> hottest

Words ending in y: change y to i

For many y-ending adjectives, swap the y before adding the ending.

noisy -> noisier -> noisiest happy -> happier -> happiest

Long words: use more / most

Long adjectives usually keep the original word and use helper words.

beautiful -> more beautiful -> most beautiful interesting -> more interesting -> most interesting
